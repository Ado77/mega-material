Mega Material

Making Elements Great Again, Material Style

Summary

An attempt at a material-design component library using custom Elements without wrapping the light-dom / BEM-styled official library. In theory they should be small and usable by any framework.

Sizes

Individual gzipped component sizes are shown below which includes both JS and CSS (all embedded in a web component). An additional 6.6Kb of "common" code contains the base element (lit-element) and the templating library (lit-html) used by all components. If your app is built using these, the components should add relatively little to your final app payload.

  • Button (1.7Kb)
  • Checkbox (2.3Kb)
  • Icon (710b)
  • Icon-button (975b)
  • Linear-progress (2.2Kb)
  • Ripple (2.6Kb)
  • Top-nav-bar (1.8Kb)
